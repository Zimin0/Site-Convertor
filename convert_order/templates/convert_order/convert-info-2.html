Как получить XML файл «новой» структуры.
1. Получить XSD-схему нового формата отчета (скачать с сайта NALOG.RU или другого органа, определяющего стандарт отчетности).
2. Определить в SAP-системе иерархию новой версии отчета (транзакция «J3RTAXREP - Иерархия системы отчетов», Меню «Новая запись»)
3. Войти в новую иерархию отчета и импортировать файл с XSD-схемой (Меню «Импорт», затем «Сохранить»)
4. Меню «Иерархии -> Экспорт SAPXML -> Сохранит файл» на локальном компьютере. Файл готов. Перетащите файл в соответствующую область сайта SAPXMLConverterUp.ru
